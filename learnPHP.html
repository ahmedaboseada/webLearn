Introduction to PHP

What is PHP?

PHP is a server-side scripting language created primarily for web development. Unlike client-side languages like JavaScript, PHP scripts run on the server. The results are sent to the client’s browser as HTML.

Characteristics of PHP

  • PHP code is executed on the server.
  • It can be integrated with databases like MySQL.
  • PHP is used in content management systems like WordPress.

How PHP Works?

PHP scripts are executed on the server and the results are sent to the client’s browser. This allows for dynamic content generation.

Syntax and Variables in PHP

Basic PHP Syntax

PHP code is executed between PHP tags, like <?php … ?>, allowing PHP integration within HTML.

Basic Example of PHP 

<?php echo "Hello, world!"; ?>

Output 

Hello, world!

Comments in PHP

Comments explain code and are ignored by PHP.

1. Single Line Comment 

<?php 
  // This is a comment
  echo "Hello World!";
?>

Output 

Hello World!

2. Multi-Line Comment 

<?php
  /* This is a multi-line comment */
  $geek = "Hello World!"; 
  echo $geek; 
?>

Output 

Hello World!

Variables and Data Types

Variables store data, declared with a $ symbol.

Declaring Variables 

<?php 
  $name = "XYZ";  
  // String
  $age = 30;       
  // Integer 
  $isEmployed = true; 
  // Boolean 
?>

Data Types

  • String: Text data.
  • Integer: Whole numbers.
  • Float: Decimal numbers.
  • Boolean: True or false.

If Conditions in PHP

What is an If Condition?

An if condition in PHP allows you to execute code only if a certain condition is true. It helps in decision-making.

Types of If Conditions:

  • Simple If: Executes code if the condition is true.
  • If-Else: Executes one block of code if the condition is true and another if it is false.
  • If-ElseIf-Else: Executes different blocks of code for multiple conditions.

PHP Example for Simple If: 

<?php
  $age = 18;
  if ($age >= 18) {
      echo "You are an adult.\n";
  }
?>

Output: 

You are an adult.

PHP Example for If-Else: 

<?php
  $score = 50;
  if ($score >= 60) {
      echo "You passed.\n";
  } else {
      echo "You failed.\n";
  }
?>

Output: 

You failed.

PHP Example for If-ElseIf-Else: 

<?php
  $grade = 85;
  if ($grade >= 90) {
      echo "Excellent\n";
  } elseif ($grade >= 70) {
      echo "Good\n";
  } else {
      echo "Need Improvement\n";
  }
?>

Output: 

Good

What is Nested If Condition?

A nested if is an if condition inside another if condition. It helps to check multiple conditions together.

PHP Example for Nested If: 

<?php
  $age = 20;
  $hasPermission = true;
  
  if ($age >= 18) {
      if ($hasPermission) {
          echo "You are allowed to enter.\n";
      } else {
          echo "Permission is required.\n";
      }
  } else {
      echo "You are not an adult.\n";
  }
?>

Output: 

You are allowed to enter.

Arrays in PHP

Arrays

  • Indexed Arrays: Array with numeric index.
  • Associative Arrays: Array with string index.
  • Multidimensional Arrays: Arrays containing other arrays.

PHP Example for Indexed Arrays:

<?php
    $fruits = array("Apple", "Banana", "Orange");
    echo $fruits[0], "\n"; // Apple
    echo $fruits[1], "\n"; // Banana
    echo $fruits[2], "\n"; // Orange
?>

Output:

Apple
Banana
Orange

PHP Example for Multidimensional Array:

<?php
    $people = array(
        array("Name" => "Zack", "Age" => 25),
        array("Name" => "Anthony", "Age" => 30),
        array("Name" => "Ram", "Age" => 22)
    );
    echo $people[0]["Name"], " is ", $people[0]["Age"], " years old.\n"; 
    echo $people[1]["Name"], " is ", $people[1]["Age"], " years old.\n";
?>

Output:

Zack is 25 years old.
Anthony is 30 years old.

Loops in PHP

What is a Loop?

A loop repeats a block of code until a condition is false. Loops help avoid repetition.

Types of Loops:

  • For Loop: For fixed number of iterations.
  • Foreach Loop: For looping through arrays.
  • While Loop: Repeat as long as the condition is true.
  • Do-While Loop: Always runs at least once before checking the condition.

PHP Example with Loops:

<?php
    // For Loop
    for ($i = 0; $i < 2; $i++) {
        echo "Hello World!\n";
    }

    // Foreach Loop
    $fruits = array("Apple", "Banana", "Orange");
    foreach ($fruits as $fruit) {
        echo $fruit . "\n";
    }

    // While Loop
    $i = 0;
    while ($i < 2) {
        echo "Hello World!\n";
        $i++;
    }

    // Do-While Loop
    $i = 0;
    do {
        echo "Hello World!\n";
        $i++;
    } while ($i < 2);
?>

Output:

Using For Loop:
Hello World!
Hello World!

Using Foreach Loop:
Apple
Banana
Orange

Using While Loop:
Hello World!
Hello World!

Using Do-While Loop:
Hello World!
Hello World!

Object-Oriented Programming (OOP) in PHP

What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm that uses "objects" and "classes" to organize code in a more structured way.

Key Concepts of OOP:

  • Class: A blueprint for creating objects (a class defines properties and methods).
  • Object: An instance of a class. It contains data and functions that define the object's behavior.
  • Encapsulation: Restricting access to some of an object's components and only exposing necessary parts.
  • Inheritance: A class can inherit properties and methods from another class.
  • Polymorphism: The ability to treat objects of different classes in a similar way.
  • Abstraction: Hiding the complexity and showing only essential features of an object.

PHP Example for Creating a Class and Object:

<?php
    class Car {
        public $brand;
        public $model;

        // Constructor to initialize properties
        public function __construct($brand, $model) {
            $this->brand = $brand;
            $this->model = $model;
        }

        // Method to display car details
        public function displayDetails() {
            echo "Brand: " . $this->brand . "\n";
            echo "Model: " . $this->model . "\n";
        }
    }

    // Create an object of class Car
    $car1 = new Car("Toyota", "Corolla");
    $car1->displayDetails();
?>

Output:

Brand: Toyota
Model: Corolla

PHP Example for Inheritance:

<?php
    class Vehicle {
        public $brand;

        public function __construct($brand) {
            $this->brand = $brand;
        }

        public function displayBrand() {
            echo "Brand: " . $this->brand . "\n";
        }
    }

    class Car extends Vehicle {
        public $model;

        public function __construct($brand, $model) {
            parent::__construct($brand);
            $this->model = $model;
        }

        public function displayDetails() {
            echo "Brand: " . $this->brand . "\n";
            echo "Model: " . $this->model . "\n";
        }
    }

    // Create an object of class Car (inherits from Vehicle)
    $car2 = new Car("BMW", "X5");
    $car2->displayDetails();
?>

Output:

Brand: BMW
Model: X5

PHP Example for Polymorphism:

<?php
    class Animal {
        public function sound() {
            echo "Animal makes a sound.\n";
        }
    }

    class Dog extends Animal {
        public function sound() {
            echo "Dog barks.\n";
        }
    }

    class Cat extends Animal {
        public function sound() {
            echo "Cat meows.\n";
        }
    }

    // Polymorphism in action
    $dog = new Dog();
    $cat = new Cat();

    $dog->sound();  // Dog barks.
    $cat->sound();  // Cat meows.
?>

Output:

Dog barks.
Cat meows.

PHP Example for Abstraction:

<?php
    abstract class Animal {
        abstract public function sound();
    }

    class Dog extends Animal {
        public function sound() {
            echo "Dog barks.\n";
        }
    }

    // Create object of Dog
    $dog1 = new Dog();
    $dog1->sound();
?>

Output:

Dog barks.

PHP with MySQL

What is PHP and MySQL?

PHP is a server-side scripting language used to interact with databases like MySQL. MySQL is a relational database management system for storing and retrieving data. PHP connects to MySQL to run queries and process data.

Basic Steps to Work with PHP and MySQL:

  1. Connect to MySQL database: Use mysqli_connect() or PDO to connect.
  2. Perform SQL queries: Use SQL to select, insert, or update data.
  3. Close the connection: Close the connection with mysqli_close() or PDO::close().

PHP Example with MySQL (Connecting, Inserting Data):

<?php
    $servername = "localhost";
    $username = "root";
    $password = "";
    $dbname = "test_db";

    $conn = new mysqli($servername, $username, $password, $dbname);
    if ($conn->connect_error) {
        die("Connection failed: " . $conn->connect_error);
    }

    $sql = "INSERT INTO users (name, email) VALUES ('John Doe', 'john@example.com')";
    if ($conn->query($sql) === TRUE) {
        echo "New record created successfully";
    } else {
        echo "Error: " . $sql . "
" . $conn->error;
    }

    $conn->close();
?>

Output:

New record created successfully

PHP Example with MySQL (Selecting Data):

<?php
    $conn = new mysqli($servername, $username, $password, $dbname);
    if ($conn->connect_error) {
        die("Connection failed: " . $conn->connect_error);
    }

    $sql = "SELECT id, name, email FROM users";
    $result = $conn->query($sql);
    if ($result->num_rows > 0) {
        while($row = $result->fetch_assoc()) {
            echo "id: " . $row["id"]. " - Name: " . $row["name"]. " - Email: " . $row["email"]. "
";
        }
    } else {
        echo "0 results";
    }

    $conn->close();
?>

Output:

id: 1 - Name: John Doe - Email: john@example.com

Important Notes:

  • MySQL Connection: Always handle connection errors to prevent failures.
  • SQL Injection Protection: Use prepared statements to avoid SQL injection attacks.
