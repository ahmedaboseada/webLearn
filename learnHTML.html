Introduction to HTML

What is HTML?

HTML (Hypertext Markup Language) is the standard language for creating web pages. It defines the structure of web pages using markup, making it the backbone of any website. HTML provides the basic structure and organizes content like text, images, and links.

Structure of an HTML Document

Document Organization

An HTML document starts with the declaration <!DOCTYPE html>, followed by the <html> tag. Inside it, we have the <head> section (for metadata) and the <body> section (for content visible to users). 

  <!DOCTYPE html>
  <html>
    <head>
      <title>Sample Page</title>
    </head>
    <body>
      <h1>Welcome to my webpage</h1>
    </body>
  </html>

Basic HTML Tags

Common Tags Used in HTML

  • <h1> to <h6>: Headings of varying levels (from largest to smallest).
  • <p>: Paragraphs of text.
  • <a>: Hyperlinks that allow users to navigate between pages.
  • <img>: Images, often used with src and alt attributes.
  • <ul> and <ol>: Unordered and ordered lists, respectively. 
  <h1>Main Heading</h1>
  <p>This is a paragraph of text. HTML is very easy to learn!</p>
  <a href="https://www.example.com">Click here</a> to visit our website.

HTML Attributes

Enhancing HTML Elements

Attributes modify the behavior or appearance of HTML elements. Common attributes include:

  • href: Specifies the URL for <a> (link) elements.
  • src: Specifies the source for <img> (image) elements.
  • alt: Provides alternative text for images.
  <a href="https://www.example.com">Visit Example</a>
  <img src="image.jpg" alt="An example image">

Semantic HTML

Meaningful Markup

Semantic HTML refers to the use of HTML tags that convey meaning about the content they enclose. Tags like <header>, <footer>, and <section> provide a better structure for web pages and improve SEO. 

  <header>
    <h1>My Website</h1>
    <nav>...</nav>
  </header>
  <footer>
    <p>Copyright Â© 2024</p>
  </footer>

HTML5 Features

New Elements and APIs

HTML5 introduced many new features such as:

  • <video>: Embeds video content without needing external plugins.
  • <audio>: Embeds audio content.
  • <canvas>: A space to draw graphics using JavaScript.
  • APIs for things like local storage, geolocation, and more. 
  <video controls>
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video tag.
  </video>

Styling with CSS

Enhancing the Look of Your Site

CSS is used to style HTML elements. For example:

  <style>
    body {
      background-color: lightblue;
    }
    h1 {
      color: darkblue;
      text-align: center;
    }
  </style>

This example changes the background color of the page and the text color of the heading.

Adding Interactivity with JavaScript

Making Your Website Interactive

JavaScript allows you to create dynamic, interactive elements on your webpage. Example: 

  <button onclick="changeText()">Click Me</button>
  <p id="demo">Hello, World!</p>
  
  <script>
    function changeText() {
      document.getElementById("demo").innerHTML = "You clicked the button!";
    }
  </script>

Creating Forms in HTML

User Input with Forms

Forms allow users to input data. Example:

  <form action="submit_form.php" method="post">
    <label for="name">Name</label>
    <input type="text" id="name" name="name"><br>
    <input type="submit" value="Submit">
  </form>

HTML Media Elements

Embedding Multimedia

HTML supports embedding various types of media like audio and video. 

  <audio controls>
    <source src="audio.mp3" type="audio/mp3">
    Your browser does not support the audio tag.
  </audio>

Creating Tables in HTML

Displaying Data with Tables

HTML tables are used to display tabular data. Example:

  <table>
    <tr>
      <th>Name</th>
      <th>Age</th>
    </tr>
    <tr>
      <td>John</td>
      <td>25</td>
    </tr>
  </table>

HTML Accessibility

Improving Website Accessibility

Accessible web design ensures your content can be used by people with disabilities. Use features like alt attributes for images, semantic tags, and proper form labeling.

Practice Examples

Here's a basic form example:

  <form>
    <input type="text" placeholder="Enter Name"><br>
    <input type="submit">
  </form>

Conclusion

HTML is the foundation of web development. With a solid understanding of HTML, you can build and structure websites. Practice with real examples, and you'll be proficient in no time!
