Introduction to JavaScript

What is JavaScript

JavaScript (JS) is a lightweight, interpreted programming language used for creating dynamic web pages. It allows developers to implement complex features such as interactive forms, animations, and real-time updates. JavaScript is supported by all modern web browsers, making it a cornerstone of web development.

Characteristics of JavaScript

  • JavaScript code is executed on the client side.
  • It can interact with HTML and CSS to create dynamic content.
  • JavaScript is essential for responsive web design and enhancing user experience.

How JavaScript Works?

JavaScript code runs in the user's web browser, allowing for real-time interaction without needing to refresh the page. This makes web applications more responsive and dynamic.

The Role of JavaScript in Web Development

JavaScript's Role in Web Development

JavaScript works alongside HTML and CSS to build modern websites:

  • HTML: Structures the content of the webpage.
  • CSS: Styles the webpage's appearance.
  • JavaScript: Adds interactivity and dynamic behavior.

For example, JavaScript enables features like dropdown menus, sliders, and pop-up alerts.

Basic Syntax of JavaScript

Basic Syntax of JavaScript

JavaScript has a straightforward syntax:

Variables

Variables are used to store data. 

                let name = "John";  // Modern syntax
                var age = 30;       // Older syntax
                const PI = 3.14;    // Constant value

Data Types

JavaScript supports numbers, strings, objects, arrays, and more. 

                let number = 10;      // Number
                let text = "Hello!";  // String
                let isOnline = true;  // Boolean

Functions

Functions are blocks of code designed to perform a specific task. 

                function greet() {
                    console.log("Hello, world!");
                }
                greet();  // Outputs: Hello, world!

Conditional Statements

Conditional statements control the flow of execution based on conditions. 

                let age = 18;
                if (age >= 18) {
                    console.log("You are an adult.");
                } else {
                    console.log("You are a minor.");
                }

Output: You are an adult.

Loops

Loops are used to repeat a block of code multiple times. 

                for (let i = 0; i < 5; i++) {
                    console.log(i);
                }

Output: 0, 1, 2, 3, 4

DOM Manipulation

DOM Manipulation

The Document Object Model (DOM) is the interface through which JavaScript interacts with the HTML structure of a webpage.

Example 

            document.getElementById("myElement").innerHTML = "Welcome!";

This code finds an HTML element with the ID myElement and changes its content to "Welcome!".

Events in JavaScript

Events in JavaScript

Events are user actions like clicks, keypresses, or form submissions. JavaScript allows developers to respond to these events.

Example 

            document.getElementById("button").addEventListener("click", function() {
                alert("Button clicked!");
            });

If Conditions in JavaScript

What is an If Condition?

An if condition in JavaScript allows you to execute code only if a certain condition is true. It helps in decision-making.

Types of If Conditions:

  • Simple If: Executes code if the condition is true.
  • If-Else: Executes one block of code if the condition is true and another if it is false.
  • If-ElseIf-Else: Executes different blocks of code for multiple conditions.

JavaScript Example for Simple If: 

<script>
            let age = 18;
            if (age >= 18) {
            document.write("You are an adult.\n");
            }
            </script>

Output: 

You are an adult.

JavaScript Example for If-Else: 

<script>
            let score = 50;
            if (score >= 60) {
            document.write("You passed.\n");
            } else {
            document.write("You failed.\n");
            }
            </script>

Output: 

You failed.

JavaScript Example for If-ElseIf-Else: 

<script>
            let grade = 85;
            if (grade >= 90) {
            document.write("Excellent\n");
            } elseif (grade >= 70) {
            document.write("Good\n");
            } else {
            document.write("Need Improvement\n");
            }
            </script>

Output: 

Good

What is Nested If Condition?

A nested if is an if condition inside another if condition. It helps to check multiple conditions together.

JavaScript Example for Nested If: 

<script>
            let age = 20;
            let hasPermission = true;

            if (age >= 18) {
            if (hasPermission) {
                document.write("You are allowed to enter.\n");
            } else {
                document.write("Permission is required.\n");
            }
            } else {
            document.write("You are not an adult.\n");
            }
            </script>

Output: 

You are allowed to enter.

Arrays in JavaScript

Arrays

  • Indexed Arrays: Array with numeric index.
  • Associative Arrays: Array with string index.
  • Multidimensional Arrays: Arrays containing other arrays.

JavaScript Example for Indexed Arrays:

<script>
            let fruits = ["Apple", "Banana", "Orange"];
            document.write(fruits[0], "\n"); // Apple
            document.write(fruits[1], "\n"); // Banana
            document.write(fruits[2], "\n"); // Orange
            </script>

Output:

            Apple
            Banana
            Orange

JavaScript Example for Multidimensional Array:

<script>
            let people = [
            {Name: "Zack", Age: 25},
            {Name: "Anthony", Age: 30},
            {Name: "Ram", Age: 22}
            ];
            document.write(people[0].Name + " is " + people[0].Age + " years old.\n"); 
            document.write(people[1].Name + " is " + people[1].Age + " years old.\n");
            </script>

Output:

            Zack is 25 years old.
            Anthony is 30 years old.

Loops in JavaScript

What is a Loop?

A loop repeats a block of code until a condition is false. Loops help avoid repetition.

Types of Loops:

  • For Loop: For fixed number of iterations.
  • ForEach Loop: For looping through arrays.
  • While Loop: Repeat as long as the condition is true.
  • Do-While Loop: Always runs at least once before checking the condition.

JavaScript Example with Loops:

                <script>
                // For Loop
                for (let i = 0; i < 2; i++) {
                    document.write("Hello World!\n");
                }

                // ForEach Loop
                let fruits = ["Apple", "Banana", "Orange"];
                fruits.forEach(fruit => document.write(fruit + "\n"));

                // While Loop
                let i = 0;
                while (i < 2) {
                    document.write("Hello World!\n");
                    i++;
                }

                // Do-While Loop
                i = 0;
                do {
                    document.write("Hello World!\n");
                    i++;
                } while (i < 2);
                </script>

Output:

                Using For Loop:
                Hello World!
                Hello World!

                Using ForEach Loop:
                Apple
                Banana
                Orange

                Using While Loop:
                Hello World!
                Hello World!

                Using Do-While Loop:
                Hello World!
                Hello World!

Object-Oriented Programming (OOP) in JavaScript

What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm that uses "objects" and "classes" to organize code in a more structured way.

Key Concepts of OOP:

  • Class: A blueprint for creating objects (a class defines properties and methods).
  • Object: An instance of a class. It contains data and functions that define the object's behavior.
  • Encapsulation: Restricting access to some of an object's components and only exposing necessary parts.
  • Inheritance: A class can inherit properties and methods from another class.
  • Polymorphism: The ability to treat objects of different classes in a similar way.
  • Abstraction: Hiding the complexity and showing only essential features of an object.

JavaScript Example for Creating a Class and Object:

<script>
            class Car {
                constructor(brand, model) {
                    this.brand = brand;
                    this.model = model;
                }

                displayDetails() {
                    document.write("Brand: " + this.brand + "
");
                    document.write("Model: " + this.model + "
");
                }
            }

            let car1 = new Car("Toyota", "Corolla");
            car1.displayDetails();
            </script>

Output:

            Brand: Toyota
            Model: Corolla

JavaScript Example for Inheritance:

<script>
            class Vehicle {
                constructor(brand) {
                    this.brand = brand;
                }

                displayBrand() {
                    document.write("Brand: " + this.brand + "
");
                }
            }

            class Car extends Vehicle {
                constructor(brand, model) {
                    super(brand);
                    this.model = model;
                }

                displayDetails() {
                    document.write("Brand: " + this.brand + "
");
                    document.write("Model: " + this.model + "
");
                }
            }

            let car2 = new Car("BMW", "X5");
            car2.displayDetails();
            </script>

Output:

            Brand: BMW
            Model: X5

JavaScript Example for Polymorphism:

<script>
            class Animal {
                sound() {
                    document.write("Animal makes a sound.
");
                }
            }

            class Dog extends Animal {
                sound() {
                    document.write("Dog barks.
");
                }
            }

            class Cat extends Animal {
                sound() {
                    document.write("Cat meows.
");
                }
            }

            let dog = new Dog();
            let cat = new Cat();

            dog.sound();  // Dog barks.
            cat.sound();  // Cat meows.
            </script>

Output:

Dog barks.
            Cat meows.

JavaScript Example for Abstraction:

<script>
            class Animal {
                sound() {
                    throw new Error("You have to implement the method!");
                }
            }

            class Dog extends Animal {
                sound() {
                    document.write("Dog barks.
");
                }
            }

            let dog1 = new Dog();
            dog1.sound();
            </script>

Output:

Dog barks.

Asynchronous JavaScript

Asynchronous JavaScript

JavaScript handles asynchronous tasks using promises and async/await syntax.

Promise Example:

            let promise = new Promise((resolve, reject) => {
                let success = true;
                if (success) {
                    resolve("Operation successful");
                } else {
                    reject("Operation failed");
                }
            });
            promise.then((message) => console.log(message));
