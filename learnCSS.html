Master the art of styling web pages using CSS.

Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. With CSS, you can control the color, font, spacing, and placement of HTML elements.

CSS Syntax

Basic CSS Syntax

CSS syntax consists of selectors and declarations. A selector targets an HTML element, and a declaration defines how to style it. 

  h1 {
    color: blue;
    font-size: 2em;
  }

CSS Selectors

Types of CSS Selectors

  • *: Universal selector (selects all elements).
  • element: Targets an element by its name (e.g., p, h1).
  • #id: Targets an element with a specific ID.
  • .class: Targets elements with a specific class.
  p {
    color: red;
  }
  .highlight {
    background-color: yellow;
  }

CSS Box Model

Understanding the Box Model

The CSS box model consists of margins, borders, padding, and the content area. Understanding this model is essential for controlling element sizing and spacing. 

  .box {
    width: 300px;
    padding: 20px;
    border: 2px solid black;
    margin: 10px;
  }

CSS Layout

Positioning and Layout

CSS provides several properties to control element positioning and layout, including display, position, and flexbox. 

  .container {
    display: flex;
    justify-content: space-between;
  }

Styling Text with CSS

Styling Text

CSS allows you to control text properties like font, size, and color.

  p {
    font-family: Arial, sans-serif;
    font-size: 16px;
    color: green;
  }

CSS Colors

Using Colors in CSS

CSS provides several ways to define colors:

  • color: red; - Named colors.
  • color: #ff0000; - Hex color codes.
  • color: rgb(255, 0, 0); - RGB color values.

CSS Backgrounds

Setting Backgrounds

You can set background images, colors, and more with CSS using the background property. 

  body {
    background-color: lightblue;
    background-image: url('image.jpg');
    background-repeat: no-repeat;
  }

CSS Transitions

Creating Transitions

Transitions allow you to animate changes in CSS properties over a set time period. 

  button {
    transition: background-color 0.3s ease;
  }
  button:hover {
    background-color: yellow;
  }

CSS Animations

Animating Elements with CSS

CSS animations let you create complex animations without JavaScript. 

  @keyframes slide {
    0% { transform: translateX(0); }
    100% { transform: translateX(100px); }
  }
  
  .animate {
    animation: slide 2s infinite;
  }

Responsive Web Design

Creating Responsive Layouts

Use media queries in CSS to create responsive designs that adapt to different screen sizes. 

  @media screen and (max-width: 600px) {
    body {
      font-size: 14px;
    }
  }

Practice Examples

Here's a simple CSS button example:

  button {
    background-color: blue;
    color: white;
    padding: 10px 20px;
    border: none;
  }
  
  button:hover {
    background-color: darkblue;
  }

Conclusion

CSS is essential for styling web pages. With a strong grasp of CSS concepts, you can design visually appealing and responsive websites.
